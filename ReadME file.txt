EMI Project ReadME file

Data Set.csv is data set file collected from Arduino Nano 32 BLE using Arduino IDE. It should be loaded for data Processing 
Main Code.ipynb is Main Jupyter NoteBook contains code for Data processing, raw data visualization of Acceleration values, Gyroscope values and Magnetometer values. it also contains particle filter code, Final visualization Output with floor plan and without floor plan.
plan R 1.jpg file that is floor plan without walking path.
plan R 4.jpg file that is floor plan with walking path in blue color.
Visualization Animation.MP4 file is a nimation of walk path trough particle filtration. 
sketch_apr26a Folder contains code for Arduino IDE to collect the Data from sensor and Beacons.
sketch_jul21a Folder contains code for ESP32 WROOM Board to run as BLE advertiser which is used as 2nd Beacon.
